# Home page script icons

Drop image files here to populate the feature cards on `/` (the home page).

Expected files:

| File | Used by | Recommended |
|------|---------|-------------|
| `lootables.png` | Lootables card | Square, 256×256 or larger, transparent background |
| `rc_cartel_heist.png` | Cartel Island Heist card | Square, 256×256 or larger, transparent background |

PNG, JPG, WebP and SVG all work. Reference them from markdown as `/images/<filename>`.
