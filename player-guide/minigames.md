# Minigames

Each box type has a unique skill-based minigame that must be completed to receive loot.

## Crate — Key Alternation

- **Objective**: Alternate pressing `A` and `D` keys to pry open the crate
- **How it works**: A progress bar fills as you press the correct keys in sequence
- **Win condition**: Reach the required number of key presses (default: 20)
- Press `ESC` to cancel

## Small Safe / Big Safe — Dial Cracking

- **Objective**: Rotate the safe dial to match 3 target numbers
- **How it works**: Use the on-screen buttons to turn the dial left or right. Match each number in the combination sequence
- **Win condition**: Successfully match all 3 numbers
- The dial has a tolerance zone — you need to be close enough to the target number
- Press `ESC` to cancel

Both safe sizes use the same minigame mechanic, just with different prop models.

## Shipping Container — Thermite

- **Objective**: Complete the thermite sequence within the time limit
- **How it works**: A thermite effect plays on the container lock. You must complete the interaction before the timer runs out
- **Win condition**: Stay in the interaction until the burn completes (default: 15 seconds)
- Press `ESC` to cancel

## Tips

- Cancelling a minigame means the box stays locked — you can try again
- If you fail, the box is not consumed — try again immediately
- The server may have modified minigame difficulty through configuration
