# Configuration

All configuration is done in two files: `config.lua` (gameplay settings) and `server/server.config.lua` (server-side security).

## Sections

- [General Settings](/rc_cartel_heist/configuration/general-settings) — debug flags, interaction mode, timers
- [NPCs & Island](/rc_cartel_heist/configuration/npcs-and-island) — guards, island environment, bunker access
- [Items](/rc_cartel_heist/configuration/items) — item names, prop spawning, melee weapons
- [Police & Dispatch](/rc_cartel_heist/configuration/police-and-dispatch) — police requirements and dispatch integration
- [Buyer](/rc_cartel_heist/configuration/buyer) — buyer NPC and sell prices
- [Server Security](/rc_cartel_heist/configuration/server-security) — database, identifier, and item whitelist
