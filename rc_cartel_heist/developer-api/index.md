# Developer API

Reference for integrating other resources with the heist. This script exposes no public exports — all cross-resource communication happens through events and `GlobalState`.

- [Server Events](/rc_cartel_heist/developer-api/server-events) — client-to-server events
- [Client Events](/rc_cartel_heist/developer-api/client-events) — server-to-client events
- [NUI Callbacks](/rc_cartel_heist/developer-api/nui-callbacks) — card minigame and debug overlay callbacks
- [GlobalState Keys](/rc_cartel_heist/developer-api/globalstate) — cross-client state synchronization patterns
