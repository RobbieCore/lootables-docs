# Items & Loot

## Heist Tools

| Item | Purpose |
|------|---------|
| EMP Blocker | Hack satellite dishes |
| Wire Cutters | Cut electrical panel wires |
| Thermite | Break container locks |
| Entrance Key | Access the island (spawns on the island) |
| Access Card | Swipe at card reader for bunker access |
